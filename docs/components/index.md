# 组件总览

## AI 相关组件

### 对话类组件
- **Bubble** - 气泡组件，用于显示对话消息
- **Conversations** - 对话列表组件
- **Sender** - 消息发送组件

### 交互类组件
- **Prompts** - 提示词组件
- **Suggestion** - 建议组件
- **Attachments** - 附件组件

### 高级组件
- **ThoughtChain** - 思维链组件
- **UseXAgent** - X Agent 组件
- **UseXChat** - X Chat 组件
- **XProvider** - X Provider 组件
- **XRequest** - X Request 组件
- **XStream** - X Stream 组件
